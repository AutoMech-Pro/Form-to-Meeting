The Logic Core

The heart of this automation is the Availability Check. The system doesn't just book blindly; it queries your calendar first. If a slot is taken, it branches to a rejection path. If free, it triggers the booking sequence.

Configuration Analysis

Building this workflow requires different levels of effort for each node. While the Trigger is simple, the Google Calendar node requires precise date formatting and OAuth2 setup. The chart below breaks down the complexity score based on setup steps (Credentials, Field Mapping, Logic Config).

1

Google Calendar: Highest complexity. Requires fetching existing events (Get Many), checking logic, and creating new events. Time formatting is critical.

2

Zoom: Moderate complexity. Requires OAuth2 and mapping the output URL to subsequent nodes.

3

Gmail & Slack: Lower complexity. Primarily mapping text fields from previous nodes to message bodies.

4

6

Notifications
